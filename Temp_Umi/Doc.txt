Aqui está uma descrição detalhada dos componentes utilizados no circuito para ler temperatura e umidade com o sensor DHT11 e o Arduino Uno:

---

 1. Arduino Uno R4
- Função: Microcontrolador principal que executa o programa e interage com o sensor DHT11.
- Características principais:
  - Baseado no microcontrolador ARM Cortex-M33.
  - Pinos de entrada e saída digital/analógica.
  - Alimentado via USB ou fonte externa.
  - Utilizado para receber, processar e exibir os dados no monitor serial.

---

 2. Sensor DHT11
- Função: Sensor que mede temperatura e umidade do ambiente.
- Características principais:
  - Gama de umidade: 20-90% com ±5% de precisão.
  - Gama de temperatura: 0-50°C com ±2°C de precisão.
  - Interface digital simples.
  - Pinos: VCC (alimentação), GND (terra), DATA (saída de dados).

---

 3. Resistor de 10kΩ (Pull-Up)
- Função: Conecta o pino de dados do DHT11 ao VCC para estabilizar o sinal digital.
- Características principais:
  - Garante uma transição estável entre níveis lógicos (0 e 1).
  - Impede flutuações no sinal de comunicação.

---

 4. Protoboard (Breadboard)
- Função: Placa de prototipagem que facilita as conexões temporárias entre os componentes.
- Características principais:
  - Permite montar o circuito sem solda.
  - Dividida em linhas de alimentação e linhas de conexão.

---

 5. Fios de Conexão (Jumpers)
- Função: Conectam os pinos do Arduino aos pinos do sensor e ao resistor.
- Características principais:
  - Coloridos para facilitar a identificação das conexões:
    - Vermelho: VCC (5V).
    - Preto: GND.
    - Amarelo: Sinal de dados.

---

 6. Computador com Arduino IDE
- Função: Interface para programar e monitorar o Arduino.
- Características principais:
  - Compila e carrega o código no Arduino via cabo USB.
  - Exibe os dados de temperatura e umidade no Monitor Serial.

---

 7. Cabo USB
- Função: Fornece alimentação ao Arduino e permite comunicação com o computador.
- Características principais:
  - Tipo USB-A para USB-C ou Micro-USB, dependendo do modelo.

---
